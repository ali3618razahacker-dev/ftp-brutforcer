This is a simple Python-based FTP brute force testing script created for learning and educational purposes only.
It attempts to log in to an FTP server using a provided username and a list of passwords to demonstrate how weak credentials can be discovered.

This project is useful for understanding:

FTP authentication behavior

Network security basics

Threading and automation in Python

Password security awareness

Features

Multi-threaded login attempts

Password list support

Fast connection testing

Stops when the correct password is found

Simple and lightweight

Requirements

Python 3.x

Built-in Python library: ftplib

No external libraries are required.
